Documents:
    Stored in JSON
    JSON - JavaScript Standard Object Notation
    {
        "key" : <value>,
    }
    Standard syntax
    MongoDB addresses problems associated with JSON (space, speed efficiency, etc)

    BSON - Binary JSON, how Mongo stores data
    
Import and Export Data:
    ("URI" -- > Uniform Resource Identifier)
    JSON Commands:
        (Import)
        mongoimport --uri "<Atlas Cluster URI>"
                    --drop=<filename>.json

        (Export)
        mongoexport --uri "<Atlas Cluster URI>"
    BSON Commands:
        (Import)
        mongostore --uri "<Atlas Cluster URI>"
                   --drop dump

        (Export)
        mongodump --uri "<Atlas Cluster URI>"
                  --collection=<collection name>
                  --out=<filename>.json

    Link Format:
        mongodb+srv://user:password@cluterURI.mongodb.net/database
